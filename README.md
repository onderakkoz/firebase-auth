## Firebase Auth

Bu proje ile kullanıcıların e-posta ve şifre ile giriş yapabilecekleri ve profil bilgilerini güncelleyip şifre değişikliği yapabilecekleri bir uygulama amaçlanmıştır.

## Kullanılan Kütüphaneler

-Firebase

-react-router-dom

-Tailwindcss

-react-redux

-@reduxjs/toolkit
